### Setup
1. Clone the project repository:
- `git clone git@github.com:Willian9857/diverse_studies.git`

2. Enter the root of the cloned project and run the commands below step by step:
- `cd diverse_studies`

3. Install the gems:
- `bundle install` or `bundle`

### Running the tests:

`rspec spec/`
